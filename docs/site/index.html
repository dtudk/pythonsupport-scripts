Python Support Scripts (macOS)¶

Minimal documentation for the macOS installation scripts and GUI.

  • macOS components: see Components
  • Orchestration and GUI: see Orchestration

Install options¶

  • GUI: run the app in MacOS/GUI/ (dummy available; real installer in progress)
  • CLI: one-liner main script
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/dtudk/pythonsupport-scripts/main/MacOS_AutoInstall.sh)"

Or locally (no curl):

bash ./MacOS/install_all.sh

Develop the docs locally¶

  • Install: pip install mkdocs
  • Serve: cd docs && mkdocs serve or use VS Code task "Docs: Serve"
  • Build: cd docs && mkdocs build (outputs to site/)
