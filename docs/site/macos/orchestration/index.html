Ways to run the installer on macOS.

CLI (one-liner)¶

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/dtudk/pythonsupport-scripts/main/MacOS_AutoInstall.sh)"

Local orchestrator (no curl)¶

bash ./MacOS/install_all.sh

GUI¶

  • Prototype GUI is available under MacOS/GUI/.
  • The GUI calls the local orchestrator and/or component scripts.
  • Current status: dummy GUI simulates steps without executing scripts.
